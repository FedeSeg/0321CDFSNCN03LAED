# Array

## Ejemplo
```
const numeros = [1,2,3,4]
const dias = ['Lunes', 'Martes', 'Miércoles', 'Jueves', 'Viernes', 'Sábado', 'Domingo']
```

## Posiciones
```
numeros[0] // primer posición
numeros[1] // segunda posición
```

## Longitud
```
numeros.length // 4
```

## Pisar posiciones
```
numeros[1] = 1.2 // cambiamos la posición 2
```
